## **cellphonedb.src.core.methods.cpdb_analysis_method**
cellphonedb.src.core.methods.cpdb_analysis_method.call(cpdb_fil_path, meta_file_path, counts_file_path, counts_data, output_path)
### **Parameters**
1. **cpdb_file_path**: `str`  
   The path to the CellphoneDB database file (e.g., `cellphonedb.zip`).
2. **meta_file_path**: `str`  
   The path to the meta file containing cell annotations.
3. **counts_file_path**: `str`  
   The path to the counts file (e.g., `counts.csv` or `h5ad` file).
4. **counts_data**: `str` (default: `'hgnc_symbol'`)  
   The column name in the counts data used for annotation (e.g., `'hgnc_symbol'`).
5. **output_path**: `str`  
   The directory path where the results will be saved.
## **cellphonedb.src.core.methods.cpdb_statistical_analysis_method()**
cellphonedb.src.core.methods.cpdb_statistical_analysis_method.call(cpdb_file_path, meta_file_path, ...)
### **Parameters**
1. **cpdb_file_path**: `str`  
   The path to the CellphoneDB database file.
2. **meta_file_path**: `str`  
   The path to the meta file.
3. **counts_file_path**: `str`  
   The path to the counts file.
4. **counts_data**: `str` (default: `'hgnc_symbol'`)  
   The column name in the counts data used for annotation.
5. **iterations**: `int` (default: `1000`)  
   The number of random permutations for statistical analysis.
6. **threshold**: `float` (default: `0.1`)  
   The minimum proportion of cells expressing a gene in a cluster.
7. **threads**: `int` (default: `5`)  
   The number of threads to use for parallel processing.
8. **debug_seed**: `int` (default: `42`)  
   A seed for reproducibility (set to `-1` to disable).
9. **result_precision**: `int` (default: `3`)  
   The number of decimal places for results.
10. **pvalue**: `float` (default: `0.05`)  
    The significance threshold for p-values.
11. **separator**: `str` (default: `'|'`)  
    The separator used to denote interactions between cell types.
12. **debug**: `bool` (default: `False`)  
    Save intermediate tables for debugging.
13. **output_path**: `str`  
    The directory path where the results will be saved.
## **cellphonedb.utils.search_utils()**
### **Import form**
```python
from cellphonedb.utils import search_utils
```
### **Parameters**
1. **query_cell_types_1**: `list`  
   A list of cell types to be paired with cell types in `query_cell_types_2` (e.g., `['EVT_1', 'EVT_2']`). Can also be `'All'` to include all cell types.
2. **query_cell_types_2**: `list`  
   A list of cell types to be paired with cell types in `query_cell_types_1` (e.g., `['PV MMP11', 'PV MYH11']`). Can also be `'All'` to include all cell types.
3. **query_genes**: `list` (optional)  
   A list of genes to filter interactions based on the genes participating in the interaction (e.g., `['TGFBR1']`).
4. **query_interactions**: `list` (optional)  
   A list of interaction names to filter interactions (e.g., `['CSF1_CSF1R']`).
5. **significant_means**: `DataFrame`  
   The `significant_means` file generated by CellphoneDB.
6. **deconvoluted**: `DataFrame`  
   The `deconvoluted` file generated by CellphoneDB.
7. **interaction_scores**: `DataFrame`  
   The `interaction_scores` file generated by CellphoneDB.
8. **query_minimum_score**: `int` (default: `50`)  
   The minimum score that an interaction must have to be included in the results.
9. **separator**: `str` (default: `'|'`)  
   The separator used to split cell types (e.g., `cellA|cellB`).
10. **long_format**: `bool` (default: `True`)  
    Converts the output into a wide table, removing non-significant interactions.
11. **query_classifications**: `list` (optional)  
    A list of classifications to filter interactions (e.g., `['Signaling by Transforming growth factor']`).
